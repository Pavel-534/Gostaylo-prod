# Инструкция для первой волны амбассадоров {brand}

Краткий гайд для приглашённых амбассадоров перед soft launch. Имя платформы в UI — из **`getSiteDisplayName()`** (env **`NEXT_PUBLIC_SITE_NAME`**, прод: **Airento**).

## Ваша главная ссылка

1. Откройте **Профиль → Реферальная программа → вкладка «Ссылка»**.
2. Скопируйте персональную ссылку или QR-код — это основной инструмент приглашений.
3. В том же разделе есть готовые тексты для Stories и мессенджеров.

## Два типа приглашений

| Кого приглашаете | Какой текст | Что происходит |
|------------------|-------------|----------------|
| **Гость / клиент** | «Для путешественников» | Бонус при регистрации по вашей ссылке |
| **Партнёр / исполнитель** | «Для исполнителей» | Host activation после **первой успешной брони** приглашённого |

Не смешивайте тексты: гостевой питч не активирует host activation.

## Если вы сами партнёр

- Разместите хотя бы **одно объявление** (`/partner/listings/new`), чтобы участвовать в программе с обеих сторон.
- На дашборде партнёра есть чек-лист «Старт как партнёр» и блок «Что делать дальше» после первого листинга.

## Где смотреть результат

- **Команда** — кто зарегистрировался по вашей ссылке.
- **Доход** — начисления и hold по правилам программы.
- **История** — движения по реферальному кошельку.

## Поддержка

При вопросах по выплатам, hold или верификации — чат поддержки в приложении или email из футера сайта.
