# Partner PDF fonts (Stage 47.0)

Bundled files for `renderPartnerFinancialStatementPdf` (Unicode: RU / TH / ZH + Latin):

| File | Source |
|------|--------|
| `NotoSans-Regular.ttf` | [Noto Sans](https://github.com/googlefonts/noto-fonts) hinted TTF |
| `NotoSansThai-Regular.ttf` | Noto Sans Thai |
| `NotoSansSC-Regular.otf` | [Noto Sans SC subset](https://github.com/notofonts/noto-cjk) (~8 MB) |

If `NotoSansSC-Regular.otf` is missing, Chinese characters in labels may render as boxes; Latin, Cyrillic, and Thai still use the first two fonts when present.
